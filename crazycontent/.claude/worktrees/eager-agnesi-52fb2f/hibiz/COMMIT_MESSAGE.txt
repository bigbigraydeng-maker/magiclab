feat: Phase 5 - Agent Pro template, Toolkit page, and Unsplash-to-Hero integration

## Summary

Complete Phase 5 implementation with three integrated features:

### A. Agent Pro Skeleton Template
- New template: "专业经纪人" (Agent Pro) for real estate agents
- Based on analysis of 3 NZ reference websites (David Ding, Drew Miller, Mike Robson)
- Modules: hero → stats-counter → listings → services → testimonials → about → faq → form → contact → footer
- 3 color palettes: deep-blue-orange, yellow-charcoal, forest-gold
- Default form template: property_valuation
- Stats counter with example data (500+ properties sold, 200+ reviews, 15+ years)

### B. Unified Toolkit Page
- New route: /app/projects/[id]/toolkit
- Consolidates all tools in single page: poster, media, social, leads, dashboard
- Merchant profile summary card (name, phone, email, logo) with edit link
- 5 tool cards with icons, descriptions, and direct links
- Mobile-first responsive design (1-col mobile, 2-col tablet, 3-col desktop)
- Auto-populated merchant profile data across all linked tools
- Simplified navigation: ProjectSubNav (项目/工具箱/数据/线索) + ProjectWorkflowSidebar (only 🧰 Toolkit)

### C. Unsplash-to-Hero Integration
- New hero_image_url field in MerchantProfileV1 (HTTPS validated)
- New Server Action: applyImageToHero() in media-actions.ts
  - Validates URL (HTTPS + safe domain)
  - Merges merchant_profile safely
  - Revalidates project/media/toolkit paths
- UX enhancements:
  - Unsplash search results: "⭐ 应用到 Hero" button for each image
  - Media library grid: "⭐ 应用到 Hero" button for saved assets
  - Project page: Hero background image preview + link to media library
- RenderMicrosite Hero priority: hero_image_url → property_promo.image_url → avatar → logo
- Published microsite: requires republish to sync hero_image_url snapshot
